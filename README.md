# 岩国駅構内タクシー 女性乗務員募集LP

## 📋 プロジェクト概要

岩国駅構内タクシーの女性乗務員募集ランディングページ（LP）です。
460px × 1200pxの縦長サイズで、駅構内デジタルサイネージやスマートフォン表示に最適化されています。

### 🎯 ターゲット
- 岩国市在住の20〜50代女性
- 主婦・子育て中の方
- 未経験からタクシー乗務員を検討している方

### 🌟 コンセプト
「タクシー＝男性の仕事」というイメージを払拭し、明るく爽やかで清潔感のあるデザインで、ママに優しい働き方をアピールします。

## 🚀 使用方法

### 1. ローカルでの確認
ブラウザで `index.html` を開くだけで動作します。

### 2. カスタマイズ箇所

#### 電話番号の変更
`index.html` の109行目を編集：
```html
<a href="tel:00-0000-0000" class="cta-button cta-button-phone">
    <i class="fas fa-phone"></i>
    <span>00-0000-0000</span>
</a>
```

#### LINE公式アカウントURLの変更
`index.html` の104行目を編集：
```html
<a href="https://line.me/R/ti/p/@YOUR_LINE_ID" class="cta-button cta-button-line" target="_blank">
```

#### 会社情報の変更
`index.html` の120-122行目を編集：
```html
<p class="footer-company">岩国駅構内タクシー</p>
<p class="footer-address">山口県岩国市</p>
```

#### メインビジュアル画像の変更
現在使用されている画像を変更する場合は、`images/hero-two-people.jpg` を置き換えてください：
```html
<div class="hero-image">
    <img src="images/hero-two-people.jpg" alt="岩国駅構内タクシー - 女性乗務員" class="hero-image-main">
</div>
```

#### ギャラリー画像の追加・変更
`images/` フォルダ内の画像を置き換えることで、ギャラリーの内容をカスタマイズできます。
画像は自動的にレスポンシブ対応されます。
